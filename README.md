## What Reel Generator Does

Reel Generator is a browser-based workflow for assembling short Instagram reel videos. It keeps the editing process centered on the essentials - trimming clips, applying filters, and preparing a final export - so you can work in one place instead of jumping between multiple apps.

The current release also provides auto caption generation to help place text on screen while you edit. Along with reference-style analysis in the workflow, the app supports building reel content from a source format or an existing visual style when that is part of your process.
